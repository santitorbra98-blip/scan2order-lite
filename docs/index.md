# Scan2Order Lite

Sistema web para digitalizacion de cartas en restaurantes con backend Laravel + frontend Vue.

## Objetivo

- Gestionar restaurantes, catalogos, secciones y productos.
- Exponer API segura con autenticacion por Sanctum.
- Servir SPA y API en un despliegue Docker unico.

## Que encontraras en esta documentacion

- Arquitectura y estructura de carpetas.
- Checklist de produccion con valores exactos para Render.
- Guia de despliegue y operacion post-deploy.
- Smoke tests automatizados para validar disponibilidad real.
