# Scan2Order Lite

Sistema web para digitalizacion de cartas en restaurantes con backend Laravel + frontend Vue.

## Objetivo del proyecto

- Gestionar restaurantes, catalogos, secciones y productos.
- Exponer API segura con autenticacion por Sanctum.
- Servir SPA y API en un despliegue Docker unico.

## Como usar esta documentacion

1. Si quieres ejecutar local: empieza por Guia rapida.
2. Si quieres desplegar: ve a Despliegue Render.
3. Si quieres validar produccion: usa Checklist + Smoke tests.
4. Si necesitas presentar el trabajo: revisa Guia para profesor.
